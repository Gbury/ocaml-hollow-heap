Functor HollowHeap.Make

module Make: 
functor (Ord : Map.OrderedType) -> S  with type key = Ord.t
Parameters:
Ord : Map.OrderedType 
type key
key is the type of keys (a.k.a. priorities): elements in the heap are ordered according to the order of their keys.
type 'a item
An 'a item is essentially an 'a. It also represents its "location" in the heap, so one can decrease its key with HollowHeap.S.decrease_key. It is an error to call decrease_key on an item which has been previously deleted by HollowHeap.S.delete_min.
type 'a t
The type of heaps. Note that these are mutable heaps.
val get : 'a item -> 'a
get xi extract the element contained in xi.
val get_key : 'a item -> key
get_key xi returns the current key associated to x in the heap. It is an error to call get_key xi if xi has been previously deleted by HollowHeap.S.delete_min.
val live : 'a item -> bool
live xi returns true if xi is still in a heap.
val create : unit -> 'a t
create () creates a new, empty, heap.
val insert : 'a t -> key -> 'a -> 'a item
insert h k x inserts x with key k in h, and returns a handle to x in h. See HollowHeap.S.decrease_key.
val merge : 'a t -> 'a t -> unit
merge h1 h2 merges the elements of h2 into h1. After merge, h2 is empty.
val find_min : 'a t -> 'a item option
find_min h returns (a handle to) the element with lowest key in h. If h is empty, returns None.
val delete_min : 'a t -> unit
delete_min h removes the element with the lowest key from h. If h is empty, delete_min h is a no-op.
val decrease_key : 'a t -> 'a item -> key -> unit
decrease_key h xi k changes the key of the element of h pointed to by xi to k. Precondition k must be no greater than the current key of xi (hence decrease).
